# shiftwise
A productivity and financial planning app designed for individuals working multiple part-time jobs.

It helps users track work hours, calculate income, manage expenses, and plan their time effectively — evolving into a smart life companion.

## 🚀 Features (Planned)
- Job & shift tracking
- Work hour logging
- Income calculation (weekly, biweekly, monthly)
- Expense tracking
- Smart reminders
- AI-based life suggestions (future)

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB

## 🎯 Vision
To become a personal companion that helps users manage work, money, and life in one place.
